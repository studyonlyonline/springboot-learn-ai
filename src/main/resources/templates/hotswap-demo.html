How Hot Swapping Works

Spring Boot DevTools automatically restarts your application when files on the classpath change.

For templates and static resources, it uses a LiveReload server to refresh your browser.

What Can Be Hot Swapped

  • Java method bodies
  • Template files (like this one)
  • Static resources (CSS, JavaScript)
  • Property files

What Requires a Full Restart

  • Adding/removing methods or fields
  • Changing method signatures
  • Adding/changing annotations
  • Changing class hierarchy
